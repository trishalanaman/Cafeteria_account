# **Cafeteria_account**
Mobile application to manage accounts in college cafeteria. Features included are as follows:
- To handle and maintain your prepaid account balance in college canteen.
- Cash free, smartphone free payment.

## Tools :
1. Kotlin
2. IntelliJ
3. Android studio
